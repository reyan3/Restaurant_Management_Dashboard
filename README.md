# 🍽️ Restaurant Management Dashboard

An interactive Restaurant Management Dashboard built using **Streamlit**, **Pandas**, and **Plotly Express** to analyze restaurant sales, customer behavior, and payment trends.

The dashboard provides actionable insights through interactive visualizations and data analysis, helping restaurant owners make data-driven decisions.

---

## 🚀 Features

- 📊 Interactive sales analysis dashboard
- 👥 Customer insights and behavior analysis
- 💳 Payment method analysis
- 📈 Dynamic charts and visualizations
- 🔍 Data filtering functionality
- ⚡ Fast and responsive Streamlit interface

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Streamlit | Web Application Framework |
| Pandas | Data Manipulation & Analysis |
| Plotly Express | Interactive Data Visualization |
| Python | Backend Logic |

## 📊 Dashboard Modules

### 📌 Overview

Provides a high-level summary of restaurant performance including:

- Monthly Revenue trajectory
- Revenue distribution by category
- Order Allocation
- Item contribution breakdown

### 📈 Sales Analysis

Analyzes:

- Monthly Revenue trends
- Top 10 revenue generating dates
- Revenue by payment method
- Category revenue heatmap

### 👥 Customer Insights

Provides insights into:

- Top 10 customer by revenue
- Customer rating distribution
- customer segmentation

### 💳 Payment Analysis

Analyzes:

- Monthly payment trends
- Payment distribution
- Revenue generated by each payment type

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/reyan3/Restaurant_Management_Dashboard
```

### 2. Navigate to the project directory

```bash
cd restaurant-management-dashboard
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit application

```bash
streamlit run App.py
```
